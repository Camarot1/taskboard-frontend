Простой проект задачника для команды/личного пользования.
Состоит из 2-х репозиториев (taskboard-frontend, taskboard-backend)
Стек Frontend: TypeScript, JavaScript, Scss.
Позволяет: создавать команды, добавять туда людей, удалять их оттуда. Добавление задач, редактирование, по Kanban таблице (Todo, In Progress, Check, Done)
Состоит из следующих страниц:
1. login - страница входа. При входе пользователю выдается токен jwt с бекенда, срок жизни токена 1 день
2.register - регистрация
3.tasks - страница просмотра задач, в выпадающем меню можно выбрать задачи которые взяли вы, и задачи команд в которых вы состоите
4.addTasks - страница добавления задач. Автоматически выбирает ваш никнейм, от вас требуется выбрать команду в которой вы состоите из выпадающего меню, заголовок и текст задачи
5.edit-task - страница редактирования задачи, в поля автоматически добавляются данные предыдущей задачи.
Модули:
1. header - шапка сайта, навигация, кнопка переключения темы сайта
2. token.tsx - модуль работы с токеном jwt. Методы:
setToken() - установка токена в localStorage при успешном логине. 
getToken() - получение токена, передаем при попытке ображения к бекенду
deleteToken() - удалить токен при выходе
decode() - разделяет токен и получает из него полезную нагрузку, не возращает ее при запросе, используется только в других функциях
usersData() - получение полезной нагрузки из токена, для использования. Содержит: id, login, companies_id. 
3. theme.tsx - НЕЙРОСЛОП! НЕ ГАРАНТИРУЮ СТАБИЛЬНУЮ РАБОТУ ИЛИ ПРОСТОТУ ПОДДЕРЖКИ! 
Записывает в localStorage состояние white или black, в зависимости от этого выводит стили на каждую страницу.
